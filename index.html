How this simulator works

Each programme already has fixed ratings from the assignment's Table 1. Your slider choices show how important each factor is. The app combines both, then estimates which programme Aiman is most likely to pick.

  • The page updates instantly when a slider changes.
  • Programme ratings can also be changed for comparison across students.
  • The chart compares all programmes side by side.
  • The preset buttons show how different student profiles lead to different outcomes.
Model assumptions and assignment notes

This interface uses the six factors required in Section B Part ii: interest, exam results, career prospects, location, fees, and willingness to explore.

  • Programme ratings are taken from Table 1 in the assignment.
  • You can edit those ratings when testing classmates' profiles for the bonus task.
  • The default importance values match Table 2 exactly.
  • The "overall score" is the model score used before converting to probabilities.
  • The probabilities are calculated using a Multinomial Logit choice model.
